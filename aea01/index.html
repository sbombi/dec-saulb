Què és Vanilla Javascript?

És JavaScript base sense cap framework o llibreria.

Diferència els ECMA Script que pertanyen a JavaScript i a JavaScript Modern.

JavaScript són ES1, ES2, ES3, ES4, ES5, ES5.1 i els JavaScript Modern són ES6, ES7, ES8, ES9, ES10.

Qui va crear Javascript? De quina empresa era?

JavaScript va ser creat per Brendan Eich de Netscape amb el nom Mocha.

Quina empresa hi ha actualment darrera de Javascript?

Cap. JavaScript és de codi obert i totalment desenvolpat per la comunitat.

Què és una arrow function? A quin ES apareix (quin any).

Les arrow functions són funcions resumides i simplificades amb característiques que les diferencien de les funcions normals (com que quan hi ha només un parametre de entrada, no fa falta posar-li parentesis) ,i van apareixer amb ES2015 (ES6).

Què és un callback? Posa un exemple.

Un callback és de manera resumida, un punter, per exemple tinc la funcio "X" i ha dins vull cridar la funció "Y".

Què és una promesa en Javascript (promise). Posa un exemple. Explica també la diferència entre la programació síncrona i la programació asíncrona, posa un exemple.

Les promeses són porcessos que están pasant mentres s'executa el codi, i poden ser cridats amb callbacks. La principal diferencia entre la programació síncrona i la asíncrona és la execució mentres que la síncrona només té un fil d'execució la asíncrona en té multiples.
